The Idea behind DoubleS paradigm

 

The idea is the virtual computer evolution to other modern machines which we call (Servers) where each modern machine is a two levels simulation machine instead of a computing machine,

 

The idea of virtual Computer evolution to Servers

 

The simulation machine has two types of simulation, the first is low level simulation where the machine (Server) simulate computer components (CPU, Memory & I/O) using units (Code Unit, Data Unit & Veto Unit) and the second type is high level simulation where the machine (Server) simulate the ideas of other programming paradigms (Structure Programming, Object-Oriented Programming, Agent-Oriented programming & Language Oriented Programming)

Define The Server?

 

       It's the basic block of programs

       The Server is a group of Units Packed together and managed to get the

          Required results

       The Server is replacement for Procedure in Structure Programming

       The Server is replacement for Class in Object Oriented Programming

       The Server is replacement for Cast in Agent Oriented Programming

 

Define The Unit?

 

   The Unit is a system plus clear interface under your request so you can interact with the unit interface to manage the unit and get the required results
