This sample show how to use a barcode font.

To install the font:-
copy the font file (3of9.TTF) to the folder windows\fonts

To Build the application:-
Run: comtemp.bat

To Run the application:-
Run: Demo.EXE

To Modify the application:-
Run: Demo.SSF (Open with PWCT)
