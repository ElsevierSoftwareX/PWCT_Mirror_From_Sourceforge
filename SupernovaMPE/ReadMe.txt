Hello World program
---------------------

1 - open command window (start - run - write cmd - enter)
2 - write notepad hello.nova
3 - write this line of code
	i want window and the window title is hello world.
    Or this code
	window title = hello world.
4 - save and exit to command window
5 - Compile and run 
	 novac hello.nova  
	 nova hello .star
    Or use the interpreter
  	 novai hello.nova
