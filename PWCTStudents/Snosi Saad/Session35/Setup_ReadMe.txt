    Create ODBC Data Source so application could access the Database
    To Create ODBC Data Source

    3.1 Open Control Panel
    3.2 Open Adminstrative Tools
    3.3 Select Data Sources (ODBC)
    3.4 Click (ADD)
    3.5 Select Driver do Microsoft Access (*.mdb)
    3.6 Data source name is MyProjectDATA
    3.7 Select Database MyProject\Data\sysdata.mdb
