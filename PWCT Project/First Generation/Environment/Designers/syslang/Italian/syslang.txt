About  Tipo Server
DoubleS Server :
Server 'Unico' nel sistema che può controllare tutto il 'DoubleS System' dall'inizio alla fine.
Super Data Server :
Server che controlla il 'Data System' nelle applicazioni DoubleS.
Super Code Server :
Server che controlla il 'Code System' nelle applicazioni DoubleS.
Super Veto Server :
Server che controlla il 'Veto System' nelle applicazioni DoubleS.
Network Server :
Server che unisce i 'System Servers' con le 'Applicazioni Servers'
Master Server :
Server 'Principale' nelle 'Applicazioni Servers'.
Slave Server :
Server che non può lavorare da solo in 'Network' senza 'Master Server'
Connection Server :
Server che connette i 'Networks' (2 o più 'Networks').
Data Server :
Server che controlla il 'Data System' in una 'Sub Network'.
Code Server :
Server che controlla il 'Code System' in una 'Sub Network'
Veto Server :
Server che controlla il 'Veto System' in una 'Sub Network'. 
Procedure Server :
Server con il solo 'Code Unit'. Lavora come 'Procedura' in Programmazione Strutturata .
Class Server :
Server senza 'Veto Unit'.Lavora come 'Class' nella Programmazione Orientata agli Oggetti.
