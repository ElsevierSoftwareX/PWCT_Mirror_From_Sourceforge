Programming Without Coding Technology (PWCT)

 

Free-Open Source software development tool for MS-Windows which you can use to create your own Visual Programming Languages (VPL) based on any text based programming languages, also PWCT Comes with general purpose Visual Programming Language ready for use by professional software developers (HarbourPWCT) .

 

 

"We open the door towards the golden age of Visual Programming Languages" .... 2010, Mahmoud Fayed

You can use PWCT to Build Console/GUI/Mixed applications  then you can distribute your applications as standalone executable files

 

Inside PWCT environment you will  Build your applications 100% without coding but you can  See, Add and/or modify the code  if you want (it's a feature - it's optional)

 

Using PWCT you will Work faster than coding by hand and you can add new components  to the environment

 

PWCT is general purpose tool and you can Build different kind of software
