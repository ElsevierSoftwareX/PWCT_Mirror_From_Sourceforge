                         Programming Without Coding Technology (PWCT)

Free-Open Source Visual Programming Language Framework for MS-Windows which you can use to create new Visual Programming Languages (VPLs) .

The project goal is developing new VPLs that are a practical alternative to Text Programming Languages (TPLs) and this done through providing a perfect solution to the common VPLs disadvantages when compared with TPLs.

PWCT comes with new programming paradigm (DoubleS = Super Server) which take in mind complex data structure, event-driven systems, client-server, object-oriented programming simulation (OOP Simulation) & visual programming languages (VPLs).  

PWCT comes with General Purpose Visual Programming Language (HarbourPWCT) ready for use by professional software developers.

PWCT comes with many samples, Order Tracking System (Database Application), and new text based programming language (Supernova programming language) developed using HarbourPWCT to clear that the HarbourPWCT visual programming language is practical general purpose visual programming language.

The current release of PWCT is 1.7 (Sharp) 2010.09.15 and the next release will be PWCT 1.8 (Smart) which will be released before the end of 2011.
