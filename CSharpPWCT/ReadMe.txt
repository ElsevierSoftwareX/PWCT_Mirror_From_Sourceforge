 ***CSharpPWCT Project***
-------------------------------------------------------------------
 		How to install this package ?
1 - copy CSharp folder to C:\SSRPWI
2 - Open PWCT-RPWI Environment only
3 - From the menu bar select Package popup
4 - Select Install Package
5 - Select the file : C:\SSRPWI\CSharp\CSHARPPWCT.PAF
-------------------------------------------------------------------
Components in this project are classified into three types :-
[1] Pure   : components wraps the language features in a direct way.
[2] Common : components uses lists to wrap more code.
[3] Quick  : components generate steps to wrap common code.
-------------------------------------------------------------------

2)		Common         c Using   	ISF,IDF & TRF	
3)		Quick	       q Using		TRF Only
