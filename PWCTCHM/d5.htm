Clearing a rectangle area, drawing a box

 

We can clear a rectangle area on the screen by filling it with space characters and using standard color setting

 

Also we can draw a box around a rectangle area.

 

Component Address:

  • Domain Name (Console Application)
  • Component Name (Basic Input/output)
