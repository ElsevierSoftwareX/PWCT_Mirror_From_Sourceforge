Introduction

 

RPWI Environment provides a group of designers to  

 

      Isolate you from the syntax of the programming language under usage

      Enable you to develop custom user interface (Human Language)

      Enable you to use this custom user interface to produce applications

 

RPWI Environment Include:-

 

      Domain Tree & Components (Installing, Reinstalling & Uninstalling)

      Goal Designer

      Interaction Designer

      Interaction Pages Generator

      Transporter Designer

      Syntax to Code Mask Converter

   Package Utility

 

Domain Tree & Components:

Manage the GUI, which is based on (Domain Tree), The RPWI Environment enables the Programmer/Developer to design custom Domain Tree, Install, Reinstall & Uninstall Components (Transporters & Servers).

 

Goal Designer:

Replacement for Code Editor, Goal Designer provides GUI Environment based on interaction instead of writing code.

 

Interaction Designer:

Used for the design of interaction pages which the programmer find when he start interaction using transporter process.

 

Interaction Pages generator

Used for quick generation of standard interaction pages

 

Transporter Designer:

Used for the design of the transporters (works as interfacing layer between both the interaction pages and the generated code)

 

Syntax to Code mask

Simple Utility takes Syntax as input, and generate code mask as output, the code mask is a simple script code used by the transporter to determine the generated code based on the data entered through interaction pages.

 

Package Utility:

Package is similar in concept to Index, it contains Custom domain tree that designed using RPWI Manager, and so you can distribute your custom domain tree.
