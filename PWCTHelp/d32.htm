Working Faster than Coding

 

Yes, this is what programmers need (Working faster than coding), at the first look to programming without coding technology (PWCT) , it seems to be very good to reduce the learning curve of programming (nice for beginners), but what about the professional programmer who used to write code very fast !!

 

Believe or not believe, programming without coding technology (PWCT) enable the professional programmer to work faster than coding while using modern code editor, how??

 

The answer is very simple, it's (Coding Simulation), where the programmer uses the goal designer as very modern code editor but without coding, simply inside the goal designer use shortcut keys and you will find your self working faster than coding , just give it a try.

 

 

CTRL+N

New File when pressed inside Server Units Window

ALT+G

Open Goal  Designer when pressed inside  Server Units Window

CTRL+R

Save, Build & Run the application

CTRL+T

Open Components Browser when pressed inside  Goal Designer

CTRL+W

Ok, save and close window (Interaction Using Transporter.)

CTRL+Y

Modify, When pressed inside Goal Designer

CTRL+N

New Step when pressed inside Goal Designer

DEL

Delete Step when pressed inside Goal Designer

CTRL+F

Form Designer when pressed inside Goal Designer

CTRL+C

Copy Steps into buffer when pressed inside Goal Designer

CTRL+V

Paste Steps from buffer when pressed inside Goal Designer

ESC

End select server, end interaction using transporter

ALT+A

Again when pressed inside Interaction Using Transporter.

A-Z , a-z

In Goal Designer(Steps Tree), open Components Browser, Send the key  *wow*, So if you know the name of the component you can get it in moments by typing the first letters of the component name
