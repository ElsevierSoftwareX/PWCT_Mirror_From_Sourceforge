About  Server Type
DoubleS Server :
the server which plays the role of the God in the system and can control all the DoubleS system from start to end.
Super Data Server :
the server which control the data system in the DoubleS application.
Super Code Server :
the server which control the code system in the DoubleS application.
Super Veto Server :
the server which control the veto system in the DoubleS application.
Network Server :
the server which link system servers with application servers.
Master Server :
the main server in the application servers.
Slave Server :
server which can't work alone in network without master server.
Connection Server :
server which connects between networks (2 or more of networks).
Data Server :
Server which control data system in sub network.
Code Server :
Server which control code system in sub network.
Veto Server :
Server which control veto system in sub network.
Procedure Server :
Only code unit so it's let the server work as procedure in structure programming 
Class Server :
Server without Veto Unit , so the server work as class in Object Oriented programming
