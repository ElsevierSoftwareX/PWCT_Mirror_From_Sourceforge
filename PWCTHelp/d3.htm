(1)            Procedure:-

 

  • Run Programming without coding technology software
  • Wait to see the logo screen
  • Click New
  • Select “Console Application – without coding”
  • Click “New”
  • Determine the file name (*.SSF)
  • Select Goal Designer
  • Click “Interact” button
  • Select Domain (Console Applications)
  • Select Component (Basic Input/Output)
  • Click ok
  • Check (Print text on new line)
  • Write the text for displaying on the screen
  • Click “ok” button
  • Select the first step
  • Click “Interact” button
  • Select Domain (Console Applications)
  • Select Component (Wait Key/Seconds)
  • Click “ok” button
  • Check “wait n seconds”
  • Determine the number of seconds to wait
  • Click ok
  • Run the program
  • Wait for saving & building the application
  • Have fun Seeing  the application in the runtime

 

This long procedure requires less than 30 seconds and it’s very easy to learn and remember, is it?
